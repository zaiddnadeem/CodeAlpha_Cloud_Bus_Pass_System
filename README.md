# 🚌 Cloud-Based Bus Pass System (CloudPass)

A responsive, dark-themed, cloud-ready **Bus Pass System** built with pure **HTML, CSS, and JavaScript** — designed as part of the **CodeAlpha Cloud Computing Internship**.

Students can register, generate a unique digital bus pass instantly, print it, or download it as a standalone HTML file.

---

## ✨ Features

- 🌑 Premium dark UI with gradient accents
- 📱 Fully responsive (mobile, tablet, desktop)
- 📝 Student registration form with validation
- 🎫 Auto-generated unique pass number (`CP-YYYY-XXXXXX`)
- 🪪 Beautiful pass card preview after submission
- 🖨 Print Pass (print-optimized CSS)
- ⬇ Download Pass as a standalone HTML file
- ☁ Ready for cloud deployment (GitHub Pages, Netlify, Vercel, AWS S3, etc.)

## 🚀 Getting Started

### Run Locally
Just open `index.html` in any browser — no build step required.

```bash
git clone https://github.com/<your-username>/bus-pass-system.git
cd bus-pass-system
open index.html
```

### Deploy to GitHub Pages
1. Push the repo to GitHub
2. Go to **Settings → Pages**
3. Select branch `main` / root
4. Your live site will be available at `https://<your-username>.github.io/bus-pass-system/`

---

## 🧪 How to Use

1. Fill the **Student Registration** form
2. Click **Generate Bus Pass**
3. View the auto-generated pass card with unique pass number
4. Use **Print** to print, **Download** to save locally, or **New Pass** to reset
